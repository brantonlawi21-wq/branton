# Chess App Documentation

## Overview
This chess app allows users to play chess online with friends or against AI.

## Features
- Online multiplayer
- AI opponent
- User profiles
- Game history

## Installation
Clone the repository and install dependencies using:
```
git clone https://github.com/brantonlawi21-wq/branton.git
cd branton
npm install
```

## Usage
Run the application using:
```
npm start
```
